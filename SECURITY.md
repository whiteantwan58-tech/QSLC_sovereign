# Security Policy

## Reporting Security Issues

**Do not** open a public issue if you discover a security vulnerability.

Instead, please email your findings to: whiteantwan58-tech@github.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Supported Versions

Security updates are provided for:
- Latest version: Always
- Previous version: Last 3 months

## Security Best Practices

- Keep dependencies updated
- Use strong authentication credentials
- Review code before merging
- Use environment variables for secrets
- Enable GitHub's security features

## Acknowledgments

Security researchers who responsibly disclose vulnerabilities will be acknowledged in our releases.

Thank you for helping keep this project secure!
